Controlando propriedades com o javascript


Com o JavaScript é possível controlar as propriedades do player como PLAY, PAUSE, STOP, MUTE E VOLUME.

No javascript pegamos o elemento pelo ID e guardamos em uma variável para manipularmos as funções


Além disso, é possível criar uma playlist para ser executada sequencialmente, trocando o arquivo para o próximo da lista.
É possível personalizar o player da forma que achar melhor, como por exemplo adicionar a barra de volume ou de execução de áudio.

Bom e por favor não coloquem música tocando quando um site abre, reproduzir um áudio deve ser uma decisão do usuário.
