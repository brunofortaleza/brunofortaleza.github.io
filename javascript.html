API de áudio da Web


A Web Audio API fornece um sistema poderoso e versátil para controlar o áudio na Web, permitindo que os desenvolvedores escolham fontes de áudio, adicionem efeitos ao áudio, criem visualizações de áudio, apliquem efeitos espaciais (como pan) e muito mais.

A API de áudio da Web envolve manipular operações de áudio dentro de um contexto de áudio e foi projetada para permitir o roteamento modular . Operações básicas de áudio são executadas com nós de áudio, que são vinculados para formar um gráfico de roteamento de áudio . Várias fontes - com diferentes tipos de layout de canal - são suportadas mesmo em um único contexto. Esse design modular oferece a flexibilidade de criar funções complexas de áudio com efeitos dinâmicos.


Gerando um beep

Vamos começar gerando um simples beep


Primeiro nós instanciamos o AudioContext, que é o principal objeto para podermos gerar um som. A partir dele nós criamos um oscilador, que é responsável por emitir um sinal contínuo, que será o nosso som.

Precisamos indicar o tipo de onde que nosso oscilador irá gerar. Indicamos que queremos uma onda senoidal (sine).

Depois conectamos o nosso oscilador ao context.destination, que representa o destino final do áudio. Se você está no computador, provavelmente isso indica as suas caixas de som. Por fim, executamos a função start() para iniciar o som.

Para parar o som, basta executar oscillator.stop().



Efeitos diferentes parando o som


Dependendo do tempo que se é usado para parar, o efeito do som pode dar uma sensação totalmente diferente.



Notas


Podemos gerar sons diferentes com determiada frequência

Para alterar a frequência, basta indicar um valor em oscillator.frequency.value. Para tocar um “Dó” (C4):

					const C4 = 261.6,
   								D4 = 293.7,
  								E4 = 329.6,
 								F4 = 349.2,
								G4 = 392.0,
								A4 = 440.0,
								B4 = 493.9;

								oscillator.frequency.value = C4;
